# ❤️ Heart Disease Prediction App

A Machine Learning based web application that predicts the likelihood of heart disease using patient health data.  
This project uses Logistic Regression for prediction and provides an easy-to-use interface for users to check risk levels.



## 🚀 Features

- Predicts possibility of heart disease
- User-friendly interface
- Machine Learning powered predictions
- Data preprocessing using StandardScaler
- Trained Logistic Regression model
- Fast and lightweight application

## 🛠️ Tech Stack

- Pandas

## 📊 Dataset

The dataset contains various medical attributes such as:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Maximum Heart Rate
- Exercise Induced Angina
- ST Depression
- And more...
